# 每日定时任务设置说明

## 快速设置

### 方法1：使用批处理脚本（推荐）

1. **右键点击** `scripts/setup_scheduled_task.bat`
2. 选择 **"以管理员身份运行"**
3. 按提示完成设置

### 方法2：手动创建

1. 打开 **任务计划程序**（搜索 "任务计划程序" 或运行 `taskschd.msc`）
2. 点击右侧 **"创建任务"**
3. 配置如下：
   - **常规** 选项卡：
     - 名称：`DailyMarketAnalysis`
     - 勾选 "使用最高权限运行"
   - **触发器** 选项卡：
     - 新建 → 每天 → 20:00
   - **操作** 选项卡：
     - 新建 → 启动程序
     - 程序：`D:\anaconda3\python.exe`
     - 参数：`D:\pythonProject2\量化交易1\Ashare复盘multi-agents\index_replay_agent\daily_task.py`
     - 起始于：`D:\pythonProject2\量化交易1\Ashare复盘multi-agents\index_replay_agent`

## 管理命令

```cmd
# 查看任务状态
schtasks /query /tn "DailyMarketAnalysis"

# 手动运行一次
schtasks /run /tn "DailyMarketAnalysis"

# 删除任务
schtasks /delete /tn "DailyMarketAnalysis" /f
```
